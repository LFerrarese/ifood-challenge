# Backend

### Construída em NodeJS, utilizando a biblioteca Express

- Métodos na pasta /api
- Arquivos JSON com os dados (sabor, tamanho, etc...) na pasta /data
- Arquivos estáticos sendo servidos na pasta /static
- Rotas contidas no arquivo /routes.js
- Inicialização do servidor no arquivo /server.js

## Como iniciar

1. Clone o projeto ou faça download da pasta com os arquivos

```
git clone https://github.com/LFerrarese/ifood-challenge.git && cd ifood-challenge/backend
```

2. Instale as dependências do projeto

```
yarn install

npm install
```

3. Inicie o servidor

```
yarn dev

npm run dev
```
