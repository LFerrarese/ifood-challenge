# Frontend

### Construído em Next.js, framework que engloba o ReactJS e adiciona novas funcionalidades como o SSR (Server-side rendering)

## Como iniciar

1. Clone o projeto ou faça download da pasta com os arquivos:

```
git clone https://github.com/LFerrarese/ifood-challenge.git && cd ifood-challenge
```

2. Instale as dependências do projeto:

```
cd frontend-next && yarn install

cd frontend-next && npm install
```

3. Instale as dependências do projeto do Backend, caso ainda não tenha feito:

```
cd backend && yarn install

cd backend && npm install
```

4. Inicie o servidor Backend:

```
yarn dev

npm run dev
```

5. Inicie o Frontend:

```
cd frontend-next && yarn dev

cd frontend-next && npm run dev
```
